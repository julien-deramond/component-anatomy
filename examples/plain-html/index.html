Anatomy

40
Root

The outermost container. Sets width context and coordinates events across all child parts.

Track

The horizontal rail representing the full value range. Acts as a positioned container for the Range and Thumb.

Range

The filled portion of the track from the minimum to the current value. Width is set as a percentage of the track.

Thumb

The draggable handle. Carries role="slider" and aria-valuenow for assistive technology.

Output

Displays the current numeric value. Uses aria-live="polite" so screen readers announce changes.
